# drawing-board

- tools : 그림판 도구 선택
  - 색상 변경
  - 굵기 변경
  - 지우개
  - 펜
  - 사각형
     테두리만 그리기 & 채워서 그리기
     정사각형
  - 원
     완벽한 원
  - 삼각형
    정삼각형
  - 텍스트 박스
  - 글자 크기 설정, 폰트 설정
- layer : 캔버스 그리기
  - 이미지 저장
  - 레이어 저장
